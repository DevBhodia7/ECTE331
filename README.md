## 🛠 Projects

### 01 · Drone Navigation
> **Fault-Tolerant Sensor Voting System**

Implements a majority-vote algorithm across redundant sensor inputs to detect and isolate faulty readings in a simulated drone navigation system. Designed for reliability under sensor failure conditions in real-time constraints.

---

### 02 · Thread Synchronization
> **Ordered Thread Execution Without Polling**

Demonstrates controlled task sequencing using semaphores and event flags, ensuring strict execution order between concurrent threads — eliminating busy-waiting and CPU waste.

---

### 03 · Robotic Arm Controller
> **Priority Inversion, Inheritance & Ceiling Protocols**

Explores classic real-time scheduling problems using a simulated robotic arm. Compares the effects of priority inversion, priority inheritance, and priority ceiling protocol on task scheduling and system determinism.
